## Migration Steps

### Step 1: Initialize the monorepo

```bash
cd /Users/niktverd/code/contento-monorepo
git init
mkdir -p apps packages/shared
git add docs
git commit -m "feat: init monorepo with docs"
```

### Step 2: Import repos WITH history

For each repo, we:
1. Add its local folder as a git remote
2. Fetch its history
3. Merge it using `--allow-unrelated-histories`
4. Move files into `apps/<original-name>/` and commit

```bash
# --- instagram-video-downloader → apps/instagram-video-downloader ---
git remote add ivd ./instagram-video-downloader
git fetch ivd
git merge ivd/main --allow-unrelated-histories --no-edit -m "merge: instagram-video-downloader history"
mkdir -p apps/instagram-video-downloader
# Move all files (except apps/, docs/, packages/) into apps/instagram-video-downloader/
git mv $(ls -A | grep -vE '^(apps|docs|packages|\.git|instagram-video-downloader|instagram-video-downloader-client|instagram-video-downloader-temporal|reals-creator-backedn|reels-creator)$') apps/instagram-video-downloader/
git commit -m "refactor: move instagram-video-downloader files to apps/instagram-video-downloader"
git remote remove ivd

# --- instagram-video-downloader-client → apps/instagram-video-downloader-client ---
git remote add ivdc ./instagram-video-downloader-client
git fetch ivdc
git merge ivdc/main --allow-unrelated-histories --no-edit -m "merge: instagram-video-downloader-client history"
mkdir -p apps/instagram-video-downloader-client
git mv $(ls -A | grep -vE '^(apps|docs|packages|\.git|instagram-video-downloader|instagram-video-downloader-client|instagram-video-downloader-temporal|reals-creator-backedn|reels-creator)$') apps/instagram-video-downloader-client/
git commit -m "refactor: move instagram-video-downloader-client files to apps/instagram-video-downloader-client"
git remote remove ivdc

# --- reels-creator → apps/reels-creator ---
git remote add rc ./reels-creator
git fetch rc
git merge rc/main --allow-unrelated-histories --no-edit -m "merge: reels-creator history"
mkdir -p apps/reels-creator
git mv $(ls -A | grep -vE '^(apps|docs|packages|\.git|instagram-video-downloader|instagram-video-downloader-client|instagram-video-downloader-temporal|reals-creator-backedn|reels-creator)$') apps/reels-creator/
git commit -m "refactor: move reels-creator files to apps/reels-creator"
git remote remove rc
# --- reals-creator-backedn → apps/reels-creator-backend (typo fixed) ---
git remote add rcb ./reals-creator-backedn
git fetch rcb
git merge rcb/main --allow-unrelated-histories --no-edit -m "merge: reals-creator-backedn history"
mkdir -p apps/reels-creator-backend
git mv $(ls -A | grep -vE '^(apps|docs|packages|\.git|instagram-video-downloader|instagram-video-downloader-client|instagram-video-downloader-temporal|reals-creator-backedn|reels-creator)$') apps/reels-creator-backend/
git commit -m "refactor: move reals-creator-backedn files to apps/reels-creator-backend"
git remote remove rcb

# --- instagram-video-downloader-temporal → apps/instagram-video-downloader-temporal ---
git remote add ivdt ./instagram-video-downloader-temporal
git fetch ivdt
git merge ivdt/main --allow-unrelated-histories --no-edit -m "merge: instagram-video-downloader-temporal history"
mkdir -p apps/instagram-video-downloader-temporal
git mv $(ls -A | grep -vE '^(apps|docs|packages|\.git|instagram-video-downloader|instagram-video-downloader-client|instagram-video-downloader-temporal|reals-creator-backedn|reels-creator)$') apps/instagram-video-downloader-temporal/
git commit -m "refactor: move instagram-video-downloader-temporal files to apps/instagram-video-downloader-temporal"
git remote remove ivdt
```

> [!WARNING]
> The `git mv` commands use a pattern to move "newly merged files" into the target subdirectory. After each merge, new files appear at the repo root — those are the ones we move. Conflicts with existing files (like `package.json`) need to be resolved by moving the incoming one into the target directory.

> [!NOTE]
> After moving files, `git log --follow apps/<folder>/somefile.ts` will still show the full history of that file from the original repo.

### Step 3: Clean up original folders

```bash
rm -rf instagram-video-downloader
rm -rf instagram-video-downloader-client
rm -rf instagram-video-downloader-temporal
rm -rf reals-creator-backedn
rm -rf reels-creator
git add -A
git commit -m "chore: remove original repo folders"
```

### Step 5: Set up root workspace

```json
// package.json (root)
{
  "name": "contento-monorepo",
  "private": true,
  "workspaces": [
    "apps/*",
    "packages/*"
  ]
}
```

```json
// packages/shared/package.json
{
  "name": "@contento/shared",
  "version": "1.0.0",
  "main": "src/index.ts",
  "types": "src/index.ts",
  "dependencies": {
    "zod": "^3.x"
  }
}
```

### Step 6: Create new GitHub repo & push

```bash
# Create niktverd/contento-monorepo on GitHub, then:
git remote add origin git@github.com:niktverd/contento-monorepo.git
git push -u origin main
```
